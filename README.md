# Moventa promotional site

Moventa is a concept landing page highlighting a modern finance app experience. The site is built with vanilla HTML, CSS, and JavaScript using a glassmorphism aesthetic.

## Deploy to GitHub Pages (5 steps)

1. **Create a repository:** Push this project to a new GitHub repository (e.g., `moventa-site`).
2. **Enable Pages:** In the repository settings, open the *Pages* section and choose the `main` branch with the `/ (root)` folder.
3. **Save settings:** Click **Save**; GitHub Pages will start building the site.
4. **Wait for publish:** After the build completes, note the published URL shown in the Pages settings.
5. **Verify content:** Visit the URL to confirm all pages (home, privacy, sitemap, robots) load correctly.
