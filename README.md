# Moventa Promo Site

Moventa is a concept mobile app focused on effortless movement tracking. This repository contains a responsive promotional landing page built with semantic HTML, modern glassmorphism-inspired CSS, and a sprinkle of vanilla JavaScript.

## Deploying to GitHub Pages (5 steps)
1. Fork or clone this repository into your GitHub account.
2. Open the repository settings and locate the **Pages** section.
3. Under **Source**, choose the `main` branch and the `/ (root)` folder, then save.
4. Wait for GitHub Pages to build the site; the deployment status appears under **Actions**.
5. Visit the published URL provided by GitHub Pages and share it with your audience.
